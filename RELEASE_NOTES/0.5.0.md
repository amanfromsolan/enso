## New
- Drag a file, folder, or image onto the terminal and its path is typed in for you — paths with spaces come out correctly escaped.
- Dropping a screenshot into Claude Code now works the way you'd expect: the image shows up in your prompt, ready to send.
- Links and text dragged from other apps are inserted at your prompt too.
- When an update is ready, a What's New panel shows exactly what changed before you install — starting with this one.
