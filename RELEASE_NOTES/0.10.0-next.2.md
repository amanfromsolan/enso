## New
- The terminal header now shows what's running: the tab's process icon appears next to the title, tinted to stay legible on light and dark terminal themes alike.
- Claude gets its face back — a new pixel-art Claude icon in the sidebar and header whenever a tab is running claude.
- What's New is one ⌘T away: open the changelog anytime from the command palette (try "Changelog" or "Release Notes").
- New tabs join the project you're in: ⌘N files the new terminal into the active tab's folder and keeps its working directory. ⌥⌘N still opens a top-level tab.
- Folders remember where they live: empty a folder out and the next terminal you open in it starts back in the project directory — even across relaunches.

## Fixed
- Sidebar process icons now track what a tab is actually running instead of latching onto the first command — and they show up on the very first command, not the second.
- Tabs created from the command palette take keyboard focus immediately; no more typing into the void.
- A collapsed folder no longer hides your active tab — it stays visible, peeking out under the folder row.
- Codex and Ollama icons no longer vanish in light mode; all sidebar iconography now adapts to the appearance.
