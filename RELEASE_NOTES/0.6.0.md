## New
- Folders can now be dragged to reorder them or move them between spaces — they tuck in while in flight and reopen where they land.
- Hovering a folder shows a + that opens a new terminal inside it, continuing in the folder's most recently used directory.
- Right-click a folder to open a new terminal in it or move it to another space.

## Improved
- Drop indicators now show exactly where things will land: between tabs, into a folder, between folders, and at the start or end of the list.
- Dragging a tab over a folder highlights the whole folder so drops read clearly.
- Open folders show a filled icon, making open and closed easier to tell apart.
- Double-click to rename now works on the name itself instead of the whole row, so clicks land where you expect.

## Fixed
- Hovering a space or folder name no longer nudges the sidebar height.
- Searching for Settings in the command palette opens the settings window again.
- Drop indicators no longer linger after the drop is done.
